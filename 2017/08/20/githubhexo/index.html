HEXO博客部署方法

发表于 2017-08-20 | 分类于 方法 | 阅读次数:

Git

HEXO

装完Git和Node.js 我们接下来要去安装HEXO

GitHub

软件准备完成后，我们现在要准备我们的博客所需要的空间。GitHub所提供的GitHub pages能让我们的静态网页有发布空间。
首先我们需要一个GitHub账号，没有的去注册一个。大家注册名称尽量不要使用大写字母。这一点的原因我将会在后面说明。
第二步 右上角选择 Create a new repository 创建一个库。名为yourname.github.io 最后创建库的结果应该为yourname/yourname.github.io。此处yourname.github.io将成为你之后的博客域名。
特别注意
这里两个yourname的名称要完全一样，包括大小写都应完全一样。因为url是不区分大小写的，为了避免强迫症发作。yourname中最好不要出现大写字母或url中不常用的字符。

进阶

快一个星期了，我总算熟悉了Hexo的一些基本操作。接下来要尽可能的对我的博客进行一些魔改。

更换链接

之前我的博客地址为 notraceoffalling.github.io这是一个基于github的二级域名。使用起来总有一种寄人篱下的感觉。下面我就将介绍如何将博客域名换成自己的专属域名。
首先去申请一个属于自己的域名。博主是在万网上申请的域名，60软妹币/年。。。。
申请完成后进入控制台->域名列表，找到你刚才注册的域名选择解析。点击添加解析按如下值添加一个解析。

保存后，我们去修改下本地文件。在{your_hexo_folder}/source/下，创建一个名字为CNAME的文件，内容即是你的个人域名。
注意：CNAME文件无后缀名
接下来重新生成发布。然后在浏览器上访问你的新域名。就可以用新域名访问你原来跑在Github服务器上的Hexo博客了。

添加回复功能

总是在博客中唱一个人的独角戏，是不是会很无聊呢。让我们来添加下评论功能吧。在网络上有许多提供商提供了评论的集成服务。
我们来盘点下
多说：已于2017年6月1日正式关停服务。 默哀。。。
Disqus:Disqus 国外服务器，由于某些原因国内无法加载。
畅言：畅言 搜狐的产品，不过网站需要备案，不然只能使用15天。
有言：有言 JiaThis旗下产品,我现在用的就是这个，操作极为简单。就是总觉得这个网站有一天会步多说的后尘。

由于NexT 5.1.0版本后已集成可评论系统，我们仅需要设置相关属性即可实现添加评论的功能。我就用博主目前正在使用的有言为例吧。
